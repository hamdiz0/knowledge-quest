## Module Information
**Module Title:** Interactive AI-Powered Educational Gaming  
**Target Audience:** Students and lifelong learners  
**Objectives:** Create an engaging learning platform that combines game mechanics with AI-generated content, promoting active learning through immediate feedback and progressive difficulty.

---

## Three Main Activities

| Activity | Specific Objective(s) | Game Mechanic | Player Type | Activity Type | Communication Type | Expected Result |
|----------|----------------------|---------------|-------------|---------------|-------------------|-----------------|
| **Answer Selection via Platform Jumping** | Master subject knowledge; develop quick decision-making | Platform jumping with arrow keys and spacebar controls | Achiever, Explorer | Individual, timed | Color-coded platforms, visual feedback  | Correct answers, knowledge retention |
| **Performance Tracking** | Self-assess progress; identify knowledge gaps | Score system with streak bonuses and time multipliers | Achiever | Data-driven | Stats dashboard, detailed analytics table | Better self-awareness, targeted improvement |
| **Progressive Difficulty** | Build confidence through gradual challenge increase | Dynamic difficulty (Easy→Medium→Hard) with adjusted time limits | Mastery-seeker | Adaptive challenge | Difficulty badges, timer adjustments | Sustained engagement, skill development |

## How AI is Used for Gamification

### 1. **Dynamic Question Generation**
- Backend integrates with Gemini API to generate custom questions on any topic
- Structured prompts ensure 7 questions with progressive difficulty levels
- Real-time content creation means unlimited quiz variations

### 2. **Personalized Learning**
- Users input any topic and receive tailored educational content
- AI adapts question complexity based on difficulty settings
- Each game session is unique and relevant to the learner

### 3. **Feedback Messages**
- Lyra the owl provides contextual encouragement based on performance
- Different messages for correct/wrong answers and streak achievements
- End-game tips personalized to accuracy and speed metrics

### 4. **Backend Architecture**
- Node.js Express server handles AI communication efficiently
- Prompt engineering ensures educational quality and proper formatting
- Fallback demo questions ensure game continuity

### 5. **Engagement Through Data and Feedback**
- Achievement badges (Bronze/Silver/Gold/Diamond) based on performance
- Detailed answer history shows questions, chosen answers, and correct answers
- Performance metrics (accuracy, streak, time) drive continued play

## Impact

By using AI for content generation, Knowledge Quest creates an endless, personalized learning experience. The platform transforms traditional quizzes into an interactive adventure where learning happens naturally through gameplay.
